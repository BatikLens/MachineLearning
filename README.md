# MachineLearning

## Dataset
This dataset consists of 10 classes of batik motifs which include:
  1. Batik Betawi
  2. Batik Cendrawasih
  3. Batik Dayak
  4. Batik Jlmaprang
  5. Batik Kawung
  6. Batik Megamendung
  7. Batik Parang
  8. Batik Sidoluhur
  9. Batik Tambal
  10. Batik Tugu Muda
      
The data (images) we collect comes from Kaggle and Google
